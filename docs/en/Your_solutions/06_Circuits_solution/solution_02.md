# Resistors Problem

We have exactly three resistors, each with resistance:

\[
1\,\Omega
\]

Using series and parallel combinations, we can form the following unique equivalent resistances.

---

## 1. All Three in Series

For resistors in series:

\[
R_{\text{eq}} = R_1 + R_2 + R_3
\]

\[
R_{\text{eq}} = 1 + 1 + 1 = 3\,\Omega
\]

---

## 2. All Three in Parallel

For resistors in parallel:

\[
\frac{1}{R_{\text{eq}}}
=
\frac{1}{1}
+
\frac{1}{1}
+
\frac{1}{1}
= 3
\]

\[
R_{\text{eq}} = \frac{1}{3}\,\Omega
\]

---

## 3. Two in Series, Then Parallel with the Third

First combine two resistors in series:

\[
1 + 1 = 2\,\Omega
\]

Now place this in parallel with the remaining \(1\,\Omega\):

\[
R_{\text{eq}}
=
\frac{2 \times 1}{2 + 1}
=
\frac{2}{3}\,\Omega
\]

---

## 4. Two in Parallel, Then Series with the Third

First combine two resistors in parallel:

\[
R =
\frac{1 \times 1}{1 + 1}
=
\frac{1}{2}\,\Omega
\]

Now place this in series with the remaining \(1\,\Omega\):

\[
R_{\text{eq}}
=
1 + \frac{1}{2}
=
\frac{3}{2}\,\Omega
\]

---

# Final List of Unique Equivalent Resistances

\[
\boxed{
\left\{
\frac{1}{3},
\frac{2}{3},
\frac{3}{2},
3
\right\}\,\Omega
}
\]

So the possible equivalent resistances are:

- \(\frac{1}{3}\,\Omega\)
- \(\frac{2}{3}\,\Omega\)
- \(\frac{3}{2}\,\Omega\)
- \(3\,\Omega\)
