Given:
R(θ) = (v₀² sin(2θ)) / g

Differentiate with respect to θ:
dR/dθ = (v₀²/g) · d/dθ[sin(2θ)]
      = (v₀²/g) · 2cos(2θ)

Set derivative to zero:
2(v₀²/g) cos(2θ) = 0

⇒ cos(2θ) = 0

⇒ 2θ = 90° + n·180°

⇒ θ = 45° + n·90°

Within physical range (0° ≤ θ ≤ 90°):
θ = 45°

Second derivative test:
d²R/dθ² = (v₀²/g)(-4 sin(2θ))

At θ = 45°:
sin(90°) = 1 ⇒ d²R/dθ² < 0 ⇒ maximum

Therefore:
Maximum range occurs at θ = 45°
